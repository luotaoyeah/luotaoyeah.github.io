## 1. 问题

在 MS SQL Server 2012 中更改表的设计并保存时出现以下提示：

![](/images/ms-sql-server-cant-save/ms-sql-server-cant-save-01.png)

## 2. 解决

取消勾选"阻止保存要求重新创建表的更改"选项；

![](/images/ms-sql-server-cant-save/ms-sql-server-cant-save-02.png)
